# TypeR - Photoshop CEP Extension

## Overview

TypeR is a Photoshop CEP (Common Extensibility Platform) extension designed for typesetters working with manga and comics scripts. It simplifies tasks like placing text on images, aligning text, and managing styles.

This is **not a standalone web application** — it is a panel extension that runs inside Adobe Photoshop. The preview in Replit will appear blank because the app depends on Adobe's CSInterface APIs which only exist inside Photoshop.

## Project Structure

- `app_src/` — Source code (React + SCSS)
  - `index.jsx` — App entry point
  - `index.html` — HTML template
  - `components/` — React components (main, textBlock, stylesBlock, modal, etc.)
  - `context.jsx` — React context/state management
  - `hotkeys.jsx` — Keyboard shortcut handling
  - `host.js` — ExtendScript (Photoshop scripting) code
  - `lib/` — Libraries (CSInterface, themeManager, JAM scripts)
  - `config.js` — App configuration
  - `utils.js` — Utility functions
- `app/` — Built output (generated by webpack)
- `themes/` — CSS theme files (dark, light, neon, etc.)
- `CSXS/` — CEP manifest for Photoshop extension registration
- `icons/` — Extension icons
- `locale/` — Localization files
- `webpack.config.js` — Webpack build configuration
- `serve.js` — Simple HTTP server for Replit preview

## Build System

- **Bundler**: Webpack 5
- **UI Framework**: React 17
- **Styling**: SCSS compiled with sass-loader
- **Transpiler**: Babel with preset-env and preset-react

## Scripts

```bash
npm install           # Install dependencies
npm run build         # Production build → app/
npm run build_dev     # Development build with source maps → app/
npm run build_watch   # Development build with file watching
```

## Replit Setup

- **Workflow**: `node serve.js` serves the built `app/` directory on port 5000
- **Deployment**: Configured as static site with `npm run build` → `app/` directory

## Installing into Photoshop

After building:

- **macOS**: `chmod +x install_mac.sh && ./install_mac.sh`
- **Windows**: `install_win.cmd`

## Key Features

- Text placement and alignment for manga/comics typesetting
- Style management with stroke (outline) support
- Auto-centering with bubble shape detection
- Customizable keyboard shortcuts
- Automatic page detection and switching
- Multiple themes (dark, light, neon, etc.)
